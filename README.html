API Endpoints

Method Endpoint Description
POST /auth/signup Register a new user
POST /auth/login Login user (returns JWT)
GET /api/posts/ Get (Paginated) Post Feed
POST /api/posts/ Create a new post
GET /api/reels/ Get (Paginated) Reel Feed
POST /api/reels/ Upload a new Reel
GET /api/stories/feed Get story feed (mutuals)
POST /api/stories/ Upload a new Story
POST /api/chats/ Create a new Chat (1-on-1 or Group)
POST /api/messages/{chatId} Send a message
DELETE /api/messages/{msgId} Delete a message
